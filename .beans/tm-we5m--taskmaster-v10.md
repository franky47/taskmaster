CLI tool for macOS and Linux that manages recurring tasks powered by Claude Code. Each task is a markdown file containing a prompt and scheduling metadata. A per-minute heartbeat evaluates which tasks are due and dispatches them, piping the prompt to Claude Code in print mode.

See [prd.md](../prd.md) for full specification.
